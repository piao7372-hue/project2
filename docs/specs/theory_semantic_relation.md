这里 eS 回答“这对图文有多大语义依据应当接近” ，C 回答“这份依据值不值得被系统采信” ，

1. 设计目标与前人基础
本节的设计目标不是再造一组复杂的相似性分支，而是用最少符号完成一个完整闭环：先由初始表
示构造关系，再对关系做可信度校准，最终得到唯一监督矩阵 S。这一路径与修正版内部方法文档
保持一致：关系由 X 决定，表示由 ChebyKAN 生成，二者不允许互换角色。在外部文献层面，本
节同时吸收了三条已被反复验证的研究线索：
• similarity-preserving 路线：跨模态哈希的核心监督对象可以直接是关系矩阵，而不是标签的简
单替代；
• local topology / manifold 路线：关系不能只看点对点，还要看局部几何和关系轮廓；
• fusion similarity / asymmetric retrieval 路线： 跨模态关系需要融合， 并且可信关系应体现双向互
认。
因此，本节不再按“高维一张、低维一张”来组织，而是按数学职责 来组织： eS 负责语义支持，
C 负责可信度校准，S 负责最终监督。

### 3.3 为什么不能只有 A
直接跨模态支持 A 只刻画当前特征空间中的瞬时配对接近性，因此它能够直接反映图像样本与文本
样本之间的点对点语义接近程度。
但是， 仅依赖A 仍然存在明显局限。 由于A 的判定基础是当前样本对之间的直接相似度， 它容
易受到单一样本表征噪声、局部偏差以及偶然高相似的影响，从而使部分样本对在缺乏稳定结构支
撑的情况下被赋予偏高的语义支持。
相比之下，R 比较的是两个样本各自在本模态中的关系轮廓，即样本在各自模态空间中的相对
结构位置是否一致。因此，R 反映的并非瞬时点对接近性，而是更稳定的结构一致性信息。
基于这一点，R 的作用不应被理解为对 A 的替代，而应被理解为对 A 的结构补偿。前者提供
直接跨模态证据，后者提供关系层面的稳定约束。二者结合后，才能形成既保留局部匹配能力、又
具备结构鲁棒性的原始语义支持矩阵 eS。

### 3.4 为什么 R 的形式是合理的
在定义上，Rij 本质上是在比较 (MI )i,: 与 (MT )j,: 这两个“关系画像”是否一致。这里，(MI )i,: 表
示图像样本 i 与全部图像样本之间的关系轮廓，(MT )j,: 表示文本样本 j 与全部文本样本之间的关系
轮廓。
因此，R 并不直接判断图像 i 与文本 j 本身是否相似，而是判断二者在各自模态空间中的 相对
接近模式是否一致。若图像 i 与一批图像样本的相对接近模式，与文本 j 与一批文本样本的相对接
近模式一致，则二者更可能处于共享语义结构中的对应位置。
这种定义方式的关键价值在于，它把 R 从单纯的公式表达提升为一个明确的结构假设：跨模态
对应样本不仅应具有直接相似性，还应在各自模态内部表现出相容的局部拓扑关系与关系轮廓。
因此，R 的构造与 local topology / manifold 路线是一致的。它试图利用模态内关系结构来补充
跨模态点对信息，从而提高原始语义支持矩阵 eS 对噪声扰动和局部失真的抵抗能力。

### 3.5 为什么 R 只能作为支持项，而不应单独充当最终监督
在本研究中，R 的定位始终应当被限定为结构补充项，而不是唯一监督来源。原因在于，R 提供的
是结构一致性证据，它能够说明两个样本是否位于共享语义结构中的相近位置，但它并不能替代直
接跨模态证据本身。
因此，将 R 单独作为最终监督矩阵并不稳妥。若过度强化 R 的作用，审稿人很可能进一步追
问：既然结构关系已足够，为什么还需要 A？这会削弱整个语义相关性模块的职责分层。
当前写法的优势恰恰在于，A 提供直接跨模态证据，R 提供结构一致性证据；前者偏局部、偏
直接，后者偏全局、偏关系；二者互补，因此采用凸组合构造原始语义支持矩阵：
eS = λA + (1 − λ)R, 0 ≤ λ ≤ 1. (6)
在这一框架下， eS 只承担“原始语义支持”的职责，而不是最终监督的全部含义。后续仍需借助可
信度矩阵 C 对支持强度进行校准，并由 S = C ⊙ eS 形成唯一的最终监督矩阵。这样既保留了 A 的
直接性，也吸收了 R 的结构性，同时避免将单一结构项误写成最终监督依据。
命题 1. eS 具有以下三点性质：
(i) 无量纲；
(ii) 元素值有界于 [0, 1]；
(iii) 既保留直接跨模态支持， 又保留结构轮廓一致性， 因此是一个原始但合理的语义支持矩阵， 而
不是最终监督矩阵。

6. 数值、量纲与工程正确性检查
为保证本节定义可直接进入实现与论文表述，这里集中给出数值与工程层面的严格检查。

### 6.1 量纲一致性
所有基础特征在式 (2) 后均为单位范数向量，因此内积、余弦、softmax 输入、概率、加权和与逐元
素乘积均为无量纲。因此，本节不存在不同量纲量直接相加或相乘的问题。

### 6.2 取值范围一致性
全节所有核心对象统一位于 [0, 1] 或 (0, 1)：

这意味着任何一步都不会把某个对象送入无界区间，从而避免了后续训练时的尺度失控。

### 6.3 非对称性与对称性的边界
MI 和 MT 是模态内 Gram 矩阵， 天然对称； 而A、R、eS、C、S 都是图像到文本的跨模态关系矩阵，
一般不要求对称。这里的非对称不是错误，而是因为行与列对应不同模态。真正需要显式考虑双向
性的地方，已经由式 (7)–(9) 完成。

### 6.4 softmax 的数值稳定性
实现时，式 (7) 与 (8) 应采用数值稳定版 softmax（如 log-sum-exp 技术）计算。由于 eSij ∈ [0, 1]，其
原始数值范围已经较窄；再配合合理的温度 τ ，指数运算整体稳定性较好。

### 6.5 复杂度与实现位置
若以全数据矩阵计算，构造 MI 与 MT 的主要代价为 O(N 2d)。在工程实现中，可在 mini-batch、
memory bank 或近邻候选集上近似计算，但 理论定义层面仍然应保持式 (4)–(10) 的完整性。也就是
说，工程近似可以改变计算策略，但不应改变对象含义。
7. 与前人工作的关系：哪些是承接，哪些是本节综合

8. 可直接放入论文的方法段落
可将本节浓缩为如下正式表述：
本文将跨模态语义相关性建模为“原始语义支持 + 可信度校准”的两阶段过程。首先，在归
一化后的初始图像特征 XI 与文本特征 XT 上，分别构造点对点的直接跨模态支持 A 与结构
对结构的轮廓支持 R，并将二者线性融合为原始语义支持矩阵 eS。其中，A 刻画跨模态样本
对的直接接近性，R 刻画跨模态样本在各自模态内部关系轮廓上的一致性，因此二者分别提
供局部直接证据与结构稳定证据。其次，为避免单向高分关系或偶然高分关系直接进入监督
阶段，基于 eS 在图像到文本与文本到图像两个方向上的条件概率，定义双向互认意义下的可
信度矩阵 C。最终，语义监督矩阵定义为 S = C ⊙ eS。其中， eS 表示“关系有多强” ，C 表示
“关系有多可信” ，而S 是后续哈希空间唯一拟合的最终关系目标。
